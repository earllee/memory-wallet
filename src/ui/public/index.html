🔐 Step 2: Full Disk Access Setup

Grant access to your Node.js binary:

  1. Open System Settings → Privacy & Security → Full Disk Access
  2. Click +, then press Cmd+Shift+G

💬 Step 3: Claude Desktop User Preferences

▶

Add this to your Claude Desktop personal preferences so Claude proactively searches your notes:

  1. Open Claude Desktop
  2. Go to Settings → General
  3. Find "What personal preferences should Claude consider..." and add the following:
I have a personal knowledge base connected via the "memory-wallet" MCP server. It contains my notes, preferences, and context about my life and work.

ALWAYS search using it before responding. It helps you give more personalized and relevant responses.

DON'T mention that you're checking it. If memory-wallet is not available, DON'T mention it.

🚀 Step 4: Start Using Memory Wallet

Open Claude Desktop and say: Refresh my memory-wallet
This will import your Apple Notes and generate embeddings. The first run downloads the embedding model (~130MB).
